{# ═══ Блок описания формулы (скрыт по умолчанию) ═══ #}
Описание расчёта
Расчётный дебит газа определяется по модифицированной формуле Гилберта-Пирвердяна, связывающей перепад давлений через штуцер (Pтр и Pшл) с расходом газа.
ФОРМУЛА (докритический режим, r < 0.5):
Q = M · (C₁ · Pтр − C₂ · Pшл) · d2 / C₃
ФОРМУЛА (критический режим, r ≥ 0.5):
Q = M · C₁ · Pтр · (1 − r) · d2 / C₃
где r = Pшл / Pтр — отношение давлений
Q — дебит газа, тыс. м³/сут
M — калибровочный множитель (по умолчанию 4.1)
Pтр — давление устья (трубное), кгс/см²
Pшл — давление шлейфа, кгс/см²
d — диаметр штуцера, мм
C₁, C₂, C₃ — эмпирические константы
Алгоритм
  1. Чтение данных давления из PostgreSQL (таблица pressure_raw)
  2. Очистка: отрицательные и нулевые значения → NaN, интерполяция (ffill/bfill)
  3. Сглаживание: фильтр Савицкого-Голея (окно 17, полином 3, 2 прохода)
  4. Расчёт дебита по формуле для каждой минутной записи (numpy-векторизация)
  5. Детекция продувок по маркерам в БД (start→press→stop) и алгоритмически по кривой давления
  6. Потери при стравливании: по уравнению Бернулли через штуцер (только фаза venting)
  7. Простои: все периоды Pтр ≤ Pшл (дебит = 0, потерь нет)
  8. Накопленный дебит: интеграция трапециями (Δt = 1/1440 суток)
Продувка vs Простой
🔥 Продувка (стравливание)
Задвижка открыта, газ стравливается в атмосферу.
Потери считаются через штуцер (Бернулли).
Определяется по маркерам (start→press) или алгоритмически.
⏱ Простой (Pтр ≤ Pшл)
Газ не поступает в шлейф. Дебит = 0.
Включает: набор давления (press→stop), тех. остановки.
Потерь нет — газ не стравливается.
Формула потерь при стравливании
Qloss = Cd · A · √(2·ΔP / ρ)
Cd = 0.9 (коэфф. расхода), A = π·d²/4 (сечение штуцера продувки), ΔP = Pуст − Pатм, ρ — плотность газа при Pуст
