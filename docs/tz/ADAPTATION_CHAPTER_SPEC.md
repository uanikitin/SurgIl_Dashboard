# Спецификация главы «Адаптация» (Chapter 4)

## Общие требования

- **Идентичность HTML и PDF**: оба рендера используют один снапшот, структура и данные совпадают
- **Источник данных**: `customer_report_block.data_snapshot` (kind = `adaptation_period_analysis`)
- **Датчики**: через `equipment_installation` → `equipment` (name LIKE 'SMOD%')

---

## Структура главы (порядок блоков)

### 1. Текстовый блок — Введение

**Содержание:**
```
В период с {date_from} по {date_to} с целью адаптации технологии
к условиям скважины №{well_number} проводились работы по выбору
оптимального реагента, оптимальной концентрации и периода дозирования.

Параметры работы скважины фиксировались в режиме реального времени
датчиками SMOD ({sensor_model}), серийные номера: {sensor_serials}.
Частота опроса — 1 измерение в минуту.
```

**Источники данных:**
| Переменная | Источник |
|------------|----------|
| `date_from` | `snap.adaptation.date_from` |
| `date_to` | `snap.adaptation.date_to` |
| `well_number` | `wells.number` (по well_id) |
| `sensor_model` | `equipment.name` WHERE name LIKE 'SMOD%' |
| `sensor_serials` | `equipment.serial_number` (список через запятую) |

---

### 2. График давлений (P устье, P линия)

**Тип:** Plotly / matplotlib (для PDF — PNG)

**Данные:** `snap.adaptation.chart_data[]`
- Ось X: `t` (timestamp)
- Ось Y1: `p_tube` (P устье, кгс/см²)
- Ось Y2: `p_line` (P линия, кгс/см²)

**События на графике:**
- Вертикальные линии: `snap.adaptation.events_for_chart[]` где `type = 'purge'`
- Точки-маркеры: `snap.adaptation.events_for_chart[]` где `type = 'reagent'`

**Пояснение под графиком:**
```
Рис. 4.1. Динамика устьевого и линейного давления за период адаптации.
Вертикальные линии — продувки, точки — вбросы реагента.
```

---

### 3. График перепада давления (ΔP)

**Данные:** `snap.adaptation.chart_data[]`
- Ось X: `t`
- Ось Y: `dp` (ΔP, кгс/см²)

**События:** аналогично п.2

**Пояснение:**
```
Рис. 4.2. Перепад давления (ΔP = P устье − P линия).
Рост ΔP указывает на увеличение дебита; снижение — на возможное
накопление жидкости или снижение притока.
```

---

### 4. График расчётного дебита (Q)

**Данные:** `snap.adaptation.chart_data[]`
- Ось X: `t`
- Ось Y: `q` (Q, тыс.м³/сут)

**События:** аналогично п.2

**Пояснение:**
```
Рис. 4.3. Расчётный дебит газа по показаниям датчиков давления.
Дебит рассчитан по формуле газодинамики с учётом диаметра штуцера
({choke_mm} мм) и свойств газа. Медиана Q = {flow_median} тыс.м³/сут.
```

**Источники:**
| Переменная | Источник |
|------------|----------|
| `choke_mm` | `snap.adaptation.choke_label` или `wells.choke_diam` |
| `flow_median` | `snap.adaptation.flow_median` |

---

### 5. Таблица событий за период

**Структура таблицы:**

| Дата/время | Тип | Описание | Кол-во |
|------------|-----|----------|--------|
| 2026-02-19 22:12 | Вброс | Реагент | 1.0 |
| 2026-02-20 08:30 | Продувка | — | — |

**Данные:** `snap.adaptation.events_for_chart[]`
- `t` → Дата/время
- `type` → Тип ('reagent' → 'Вброс', 'purge' → 'Продувка')
- `label` / `reagent` → Описание
- `qty` / `amount` → Кол-во

**Итоговая строка:**
```
Всего: {reagent_count} вбросов реагента ({reagent_qty} ед.), {purge_count} продувок.
```

---

### 6. Плитки статистического анализа (4 карточки)

**Карточки:**

| Карточка | Цвет | Источник |
|----------|------|----------|
| R (Адаптация) | Оранжевый | `snap.adaptation` |
| R★ (Оптимум) | Зелёный | `snap.optimal` |
| B2 (Наблюдение) | Синий | `snap.observation` |
| B1 (Базовый) | Серый | `snap.b1` |

**Поля каждой карточки:**

| Поле | Ключ в снапшоте | Единица |
|------|-----------------|---------|
| Q мед. | `flow_median` | тыс.м³/сут |
| Q ср. | `flow_avg` | тыс.м³/сут |
| ΔP мед. | `dp_median` | кгс/см² |
| P труб. | `p_tube_median` | кгс/см² |
| P лин. | `p_line_median` | кгс/см² |
| КИВ | `utilization_pct` | % |
| Дней | `duration_days` | сут. |

---

### 7. Таблица сравнения (Наблюдение → Адаптация)

**Структура:**

| Параметр | B2 (Набл.) | R (Адапт.) | Δ | Δ% |
|----------|------------|------------|---|-----|
| Q медианный | 9.01 | 19.86 | +10.85 | +120.4% |
| Q средний | 8.59 | 19.35 | +10.76 | +125.3% |
| ΔP медиана | 0.40 | 1.96 | +1.56 | +390.0% |
| P труб. медиана | 17.30 | 19.07 | +1.77 | +10.2% |
| P лин. медиана | 16.90 | 17.10 | +0.20 | +1.2% |
| КИВ, % | 75.00 | 99.78 | +24.78 | — |
| Продувок | 0 | 0 | 0 | — |
| Вбросов реагента | 2 | 18 | +16 | — |

**Данные:**
- Колонка B2: `snap.observation.*`
- Колонка R: `snap.adaptation.*`
- Колонки Δ, Δ%: `snap.comparison.delta_*`, `snap.comparison.delta_*_pct`

---

### 8. Тренды за период адаптации

**Формат:**
```
📈 Тренды за период адаптации:
Q: ↓ -0.159 тыс.м³/сут/день (знач.)
ΔP: ↓ -0.0124 кгс/см²/день
P труб: ↓ -0.172 /день
P лин: ↓ -0.128 /день
```

**Данные:**
| Тренд | Источник |
|-------|----------|
| Q | `snap.adaptation.q_trend` |
| ΔP | `snap.adaptation.dp_trend` |
| P труб | `snap.adaptation.p_tube_trend` |
| P лин | `snap.adaptation.p_line_trend` |

Каждый тренд содержит: `direction`, `slope_per_day`, `mk_significant`

---

### 9. Сегментный анализ

**Источник:** отдельный блок `segment_analysis` или встроенный в `adaptation_period_analysis`

#### 9.1. График сегментов Q

**Тип:** Plotly / matplotlib

**Данные:** `snap.segment_analysis.chart_data`
- `dates[]` — временные метки
- `q_total[]` — значения дебита

**Changepoints:** `snap.segment_analysis.changepoints[]` (индексы точек разрыва)

**CP Marks:** `snap.segment_analysis.cp_marks[]`
```json
{"idx": 16, "tag": "CP1", "date": "2026-02-02 19:00", "source": "only_total"}
```

**Визуализация:**
- Линия Q с цветовой разбивкой по сегментам
- Вертикальные пунктирные линии на changepoints
- Метки CP1, CP2, ... над точками разрыва

**Пояснение:**
```
Рис. 4.4. Сегментный анализ расчётного дебита.
Автоматическое выделение участков стабильной работы и переходных процессов.
Вертикальные линии — точки изменения режима (changepoints).
```

---

#### 9.2. Описание сегментов

**Данные:** `snap.segment_analysis.interpretation.descriptions[]`

**Формат:**
```
Сегмент 1: initial, среднее 2.14 тыс.м³/сут
Сегмент 2: sharp_decline, среднее 10.95 тыс.м³/сут
Сегмент 3: sharp_rise, среднее 1.19 тыс.м³/сут
...
```

**Типы сегментов:**
| Тип | Описание |
|-----|----------|
| `initial` | Начальный |
| `stable` | Стабильный |
| `rise` | Рост |
| `decline` | Снижение |
| `sharp_rise` | Резкий рост |
| `sharp_decline` | Резкое снижение |

---

#### 9.3. Сводка сегментного анализа

**Данные:** `snap.segment_analysis.interpretation.summary`

**Пример:**
```
За анализируемый период выделено 11 сегментов:
Начальный — 1, Резкое снижение — 7, Резкий рост — 2, Рост — 1.
Всего зафиксировано 1 вбросов реагента.
Общая динамика за период: среднее значение выросло на 725.9% (с 2.14 до 17.65).
```
