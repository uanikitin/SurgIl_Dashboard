# HANDOFF — Паритет HTML-главы и PDF (плитки + графики), 2026-06-21

Цель пользователя: PDF должен выглядеть как HTML-глава — **метрики плитками**
(а не таблицами), графики максимально похожи. Не пиксель-в-пиксель, но «тот же отчёт».

## Реальность
HTML = браузер + CSS-плитки + Plotly. PDF = LaTeX + matplotlib. Идентичность
пиксель-в-пиксель недостижима; цель — визуальная консистентность.
